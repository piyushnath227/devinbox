Real Qwen Tool-Use

The agent calls search_repo and read_file mid-reasoning to inspect actual code before proposing a fix, instead of guessing from the issue text alone.

Human-in-the-Loop

Issues are classified, diffs are generated, and draft PRs are opened automatically — but nothing merges without an explicit /approve.

Durable Audit Trail

Every classification, diff, and PR decision is logged and optionally archived to Alibaba Cloud OSS, with a webhook idempotency guard against duplicate deliveries.

Works on Repos You Don't Own

If DevInbox lacks direct write access, it automatically forks the repository and opens a cross-repo pull request back to upstream — the same workflow a human open-source contributor would use.
